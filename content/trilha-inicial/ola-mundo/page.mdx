# Olá Mundo

Bem-vindo à sua primeira aula na **Trilha do Programador**. Hoje vamos aprender o clássico "Hello World".

## O que é um programa?

Um programa é uma sequência de instruções que o computador executa.

## Exemplo em JavaScript

```javascript
console.log("Olá, Mundo!");
```

## Exemplo em Python

```python
print("Olá, Mundo!")
```

Parabéns! Você acabou de ver seu primeiro código.
